공간 padding 여백 margin

            padding(공간) - 내가 가진 범위 틀 안에서 여유있는 공간을 설정하는 것
                pt- : padding-top 태그 위쪽 패딩을 설정
                pr-: padding-right 태그 오른쪽 패딩을 설정
                pb-: padding-bottom 태그 아래쪽 패딩을 설정
                pl-:padding-left 태그 왼쪽 패딩을 설정

                px-:태그의 왼쪽과 오른쪽 패딩을 설정
                py-:태그의 위쪽과 아래쪽 패딩을 설정

                p-:태그의 모든 방향 패딩을 설정

            Margin(여백)- 다른 태그나 다른 값과 거리를 두는 것
 mt- : padding-top 태그 위쪽 마진을 설정
                mr-: padding-right 태그 오른쪽 마진을 설정
                mb-: padding-bottom 태그 아래쪽 마진을 설정
                ml-:padding-left 태그 왼쪽 마진을 설정

                mx-:태그의 왼쪽과 오른쪽 마진을 설정
                my-:태그의 위쪽과 아래쪽 마진을 설정

                m-:태그의 모든 방향 마진을 설정

숫자값

0,1,2,3,4,5, auto를 사용할 수 있음 0: 여백이나 패딩이 없음 1: 25rem(=4px) 2: 5rem(=8px) 3: 1rem (=16px) 4: 1.5rem (=24px) 5: 3rem (=48px) rem 과 px 관계 0.25rem 0.25*16px= 4px 0.5rem 5*16px= 4px 1.rem 1*16px= 4px 1.5rem 1.5*16px= 4px 3rem 3*16px= 4px
