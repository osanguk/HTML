부트스트랩(bootstrap)

            트위터에서 개발한 오픈 소스 프론트엔드 프레임워크
            반응형 웹 디자인


            부트스트랩은 프레임워크 이기 때문에 어느정도 기본 스타일은 제공하나
            추가적인 스타일이 필요할 경우 css 를 사용해서 디자인을 만들어줘야 함

            부트스트랩으로 이동

            사용법
            
           부트스트랩 js 적용하기
           

           integrity 이란?
            -링크를 안전하게 가져오기 위해 설정하는 속성 (필수는 아니지만 보안, 신뢰성을 위해 사용)
            crossorigin 이란?
            -anonymous 자격 증명을 포함하지 않도록 설정  (필수는 아니지만 보안, 신뢰성을 위해 사용)

라이브러리와 프레임워크의 차이 

            라이브러리vs 프레임워크

            라이브러리 = 샤브샤브뷔페
            ->개발자가 개발 하는데 있어 사용범위가 더 넓음
                
            프레임워크 = 샤브샤브밀키트
            ->개발자가 개발 하는데 있어 사용범위가 제한적임
