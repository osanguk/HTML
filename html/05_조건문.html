조건문

            프로그래밍

if(만약에) 문

            
                if(조건식){
                    조건식이 true일 경우 실행할 코드;
                }
            
            * 조건식 : 결과가 true, false인 식

if-else 문

            
                if(조건식){
                    조건식이 true인 경우 수행할 코드;
                }else{
                    조건식이 false인 경우 수행할 코드;
                }
            

            *else는 조건식을 작성하는 ()없음
0~100 사이에 랜덤 숫자를 발생시켜서 홀/짝 판별
