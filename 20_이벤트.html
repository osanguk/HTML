이벤트(event)

            *이벤트 : 동작,행위 -> 브라우저에서 할 수 있는 동작

            ex)click,keydown,keypress,keyup,mouseover.....
            ->위와 같은 방법은 id 작성 후 script 코드 작성
            button.addEventListener("click",function(){
                alret("버튼이 클릭되었습니다.")
            })

이벤트 리스너(Event Listner)

            브라우저(웹 화면)가 이벤트가 감지 될 때 까지 대기하다가
            감지가 되었을 때 연결된 기능(함수)를 수행하는 것

            -이벤트 리스너는 on이벤트명 형식으로 작성됨

            ex) onclick, onchange, onsubmit, onkeyup ....
            function 기능명(){
                alret("버튼이 클릭되었습니다.");
            }
