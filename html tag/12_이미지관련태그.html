이미지 태그

            -웹 문서에 사진이나 그림 같은 이미짖를 삽입하는 용도의 태그

            [속성]
            (속성 : 태그에 작성하는 내용)

            1. src : 삽입할 이미지의 경로를 작성하는 속성
                (파일 위치,인터넷 주소 = 이미지를 가져올 수 있는 경로)

                2.width : 가로길이를 지정하는 속성(필수x)
                3.height:  세로길이를 지정하는 속성(필수x)
                4.alt : 이미지 설명을 작성하는 속성
                ->이미지가 보이지 않을 때 하면에 출력되는 그림

                코드에서 뒤로가기를 작성할 때에는 ../ 를 이용하여 뒤로가기를 진행한다.
                뒤로가기를 3번 진행하고 싶으면 ../ ../ ../ 를 하면 된다.

상대 경로 이미지 출력

            상대경로 : 현대 파일/페이지 위치를 기주느오 하여
            목표까지의 경로를 나타낸 것
            (파일 기준에서부터 시작하는 위치)
            (학원에서 엘리베이터를 타고 1층으로 내려가서 오르쪽에 세븐일레븐이 있음)
