            가장 많이 사용하는 태그

         div 옆에 작성한 class는 head부터 foot 까지 모두 사용 가능
         p h1 pre table 등 모든 태그에 사용 가능

inline 형식

            -화면에서 가로를 사용
            (가로화면을 세로로 잘라서 나눔)

            -한 층을 모두 지지하지 않음
            ---내부에 작성된 내용 크기 만큼 차지
            -width, height 와 같이 크기 지정 속성 불가
            -line 형식 태그
            --- b(굵기)
            ---i(기울이기)
            ---ㅡmark(글 배경 색)
            -- span(영역 분할)등

in line 형식은 가로를 세로로 분할하는 성질을 가지고 있어서 줄이 바뀌지 않고 계속 옆으로 작성된다.
